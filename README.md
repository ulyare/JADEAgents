Основные файлы проекта лежат в папке src->jade. Запуск кода осуществляется из файла MainContainer.java. 
Проект создавался в IntelliJ IDEA, структура проекта сохранена. 
В папке agents прикреплены две фотографии, на одной тополгия графа и ручной проход алгоритма по этапам, на второй рассчёт сложности.

Описание алгоритма. На первом этапе агенты знают только своё число и свои связи с другими агентами. На втором этапе каждый агент передаёт всем доступным агнетам имеющиеся данные. На третьем этапе аналогичные действия. И так до тех пор пор, пока все агенты не узнают обо всех. После этого каждый агент вычисляет среднее арифметическое.
